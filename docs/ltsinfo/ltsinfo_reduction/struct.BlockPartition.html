A partition that explicitly stores a list of blocks and their indexing into the list of elements.

Create an initial partition where all the states are in a single block 0. And all the elements in the block are marked.

pub fn partition_marked_with<F>( &mut self, block_index: usize, builder: &mut BlockPartitionBuilder, partitioner: F, ) -> impl Iterator<Item = usize>
where F: FnMut(usize, &BlockPartition) -> usize,

Partition the elements of the given block into multiple new blocks based on the given partitioner; which returns a number for each marked element. Elements with the same number belong to the same block, and the returned numbers should be dense.

Returns an iterator over the new block indices, where the first element is the index of the block that was partitioned. And that block is the largest block.
